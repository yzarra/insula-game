## HOW TO PLAY (.jar) - recommended
1. Download the .jar file (latest one recommended)
2. Double-click the .jar, and it should run just like that!
* note: If your computer does not let you run the .jar file,
  Download the .txt file and change its name to end in .jar once downloaded
* You can also run it through the terminal if you're having issues with this command:

  java -jar DressingRoom.jar
* note: the jar already includes the assets file so you're good to go!

## HOW TO PLAY (through IDE) - for development
1. Import the .java files into your Java IDE of choice inside the src file
2. For now, the IDE expects the assets folder to be located at "C:\gdxprojects\crying-2\assets\", so put it there
3. Run through driver as usual :)
