# Insula – Character Customization Game
* A cross-platform 2D character customization game with downloadable builds and a browser version.
* Built in Java (Swing) and then ported to Python (Pygame) and Javascript/HTML/CSS

## Play / Download ##
You can download on Windows and Mac! Or just play on your browser

All available here:

https://github.com/yzarra/insula-game/releases/tag/insula

## Project Structure ##
- `/java` → Java version
- `/python` → Python version
- `/docs` → Web (JS, HTML, CSS) version

Each version has its own setup instructions in its respective folder.
